### Steps

<hr/>

**Step 1**

> <article>1. Run the following command and note the value output "mmctl config get FileSettings.MaxFileSize"<br>2. Run the following command to change the value "mmctl config set FileSettings.MaxFileSize 104857600"<br>3. Run "mmctl config get FileSettings.MaxFileSize" again to check the new value</article>

**Expected**

> <article>After 2) the command should exit correctly<br>After 3) the value returned should be the one set in step 2). </article>
