## MM-T4965: JWT settings are not shown when the license is set to Cloud

### Details

| Name     | Value                    |
| :------- | :----------------------- |
| Priority | Normal                   |
| Status   | Draft                    |
| Folder   | Plugins / Zoom / General |
| Created  | 2022-08-08T14:43:30Z     |

### Steps

<hr/>

**Step 1**

> <article>1. As a sysadmin - upload a cloud license<br />2. Navigate to plugin settings for Zoom</article>

**Expected**

> <article>- Enable oAuth is <strong>not</strong> shown (always true)<br />- API Key and API Secret are <strong>not</strong> shown</article>

**Step Info**
| Name | Value |
| :--- | :--- |
| Important notes about this step | This test requires an oAuth app on the Zoom side that uses Account-level app<br /><br />See steps here https://mattermost.gitbook.io/plugin-zoom/installation/zoom-configuration/zoom-setup-oauth |

<hr/>

**Step 2**

> <article>1. As a sysadmin - remove the Cloud license<br />2. Navigate to plugin settings for Zoom</article>

**Expected**

> <article>- Enable oAuth visible<br />- API Key and API Secret are shown</article>

**Step Info**
| Name | Value |
| :--- | :--- |
| Important notes about this step | This test requires an oAuth app on the Zoom side that uses Account-level app<br /><br />See steps here https://mattermost.gitbook.io/plugin-zoom/installation/zoom-configuration/zoom-setup-oauth |
