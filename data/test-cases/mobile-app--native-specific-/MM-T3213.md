### Steps

<hr/>

**Step 1**

> <article>Should be able to create a GM channel and post a message to the group<br>--------------------<br>1. Login to server and open main sidebar<br>2. Tap on plus button (<strong>+</strong>) next to <strong>Direct Messages</strong><br>3. Type some letters to filter the user list<br>4. Tap to select 2 or more users (don't tap your own user first)<br>5. Search for your own user<br>6. Tap <strong>Start</strong> button<br>7. Post a message in the GM channel</article>

**Expected**

> <article>After #5,<ul><li>Verify your own user is not displayed and cannot be add to a GM</li></ul><br>After #6,<ul><li>GM channel is created and displayed</li></ul><br>After#7,<ul><li>Message is posted in the GM channel</li></ul></article>
