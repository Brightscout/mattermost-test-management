### Steps

<hr/>

**Step 1**

> <article>Post a file attachment<br>————————————————————————————<ol><li>Navigate to the channel you want to upload the file in</li><li>Tap the icon to attach a file or the icon to attach an image (not the camera icon, but you can select a photo file if you like)</li></ol></article>

**Expected**

> <article><ul><li>File selection on your device should open</li></ul></article>

<hr/>

**Step 2**

> <article><ol><li>Select a file to upload</li><li>Observe the loading indicator (if it doesn't upload immediately)</li></ol></article>

**Expected**

> <article><ul><li>The file should start uploading, loading indicator displays if needed, and then a preview should be displayed</li></ul></article>

<hr/>

**Step 3**

> <article><ol><li>Type a message (if desired) and tap the aeroplane icon to send.</li></ol></article>

**Expected**

> <article><ul><li>Message and file attachment should be posted.</li></ul></article>
