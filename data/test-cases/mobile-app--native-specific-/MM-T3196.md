### Steps

<hr/>

**Step 1**

> <article>Add members to channel<br>————————————————————————————<ol><li>Tap a channel title to view channel info, and note the number of channel members next to Manage Members</li><li> Tap on "Add members", observe list of users on the server</li><li> Tap on "Search", observe keyboard opens</li><li> Type in the name of the user to be added, observe user list filters accordingly</li><li> Tap on the circle to the left of the user's name (first tap may just close keyboard; may have to tap again), observe the circle next to their name appears selected</li><li>Tap on "add" at the top right of the screen</li></ol></article>

**Expected**

> <article><ul><li>The user is added to the channel</li><li>Total number of members next to "manage members" menu option should increase by the number of users added to the channel</li><li>System message should display in the channel showing which members were added to the channel.</li><li>Tapping on the user's name in the system message should open the user's profile from where you can send them a DM</li></ul></article>
