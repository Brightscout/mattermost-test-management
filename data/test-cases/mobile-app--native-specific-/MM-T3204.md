### Steps

<hr/>

**Step 1**

> <article>Add user to private channel<br>————————————————————————————<ol><li>Tap the name of a private channel where you have permission to add members</li><li>Tap `Add Members`</li><li>Type the beginning of a user's name in the search box</li><li>Select user</li><li>Tap Add</li></ol></article>

**Expected**

> <article>After #2,<br><ul><li>A list of users on the server should be displayed</li></ul>After #3,<br><ul><li>Your keyboard should pop up so you can type in the name of a user.</li><li>The list should filter as you type in the name and display all users matching your input</li></ul>After #4,<br><ul><li>The circle should show blue with a white tick (selected)</li></ul>After #5,<br><ul><li>The user should be added to the channel, total number of members next to "manage members" menu option should increase by the number of users added to the channel and a system message should display in the channel showing which members were added to the channel.</li></ul><br><br></article>
