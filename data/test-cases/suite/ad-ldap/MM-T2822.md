### Steps

<hr/>

**Step 1**

> <article>Change/Remove LDAP Admin Filter<br>————————————————————————————<ol><li>Ensure LDAP setup is done correctly on an instance</li><li> Navigate to System Console ➜ AD/LDAP and set Admin Filter to "(givenName=barrett)"</li><li> Login as a LDAP user who has this attribute "(givenName=barrett)" configured in the AD/LDAP Server</li><li>Now login as sysadmin and set Enable Admin Attribute to false and check</li></ol></article>

**Expected**

> <article><ul><li>Existing LDAP System Admin users should not be demoted to members.</li><li>Any new LDAP user who tries to login with "givenName=barrett" will no longer be considered a System Admin user and instead should be considered a regular member</li></ul></article>
