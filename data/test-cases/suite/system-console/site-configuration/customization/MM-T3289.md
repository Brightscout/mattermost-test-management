## MM-T3289: Help (Ask community link setting)

### Details

| Name     | Value                                                       |
| :------- | :---------------------------------------------------------- |
| Priority | Normal                                                      |
| Status   | Active                                                      |
| Folder   | Suite / System Console / Site Configuration / Customization |
| Labels   | system-console                                              |
| Created  | 2020-08-07T20:25:06Z                                        |

### Steps

<hr/>

**Step 1**

> <article>Enable Ask community link<ol><li>Navigate to: System Console ➜ Site Configuration ➜ Customization</li><li>Scroll to Enable Ask Community Link section (above Help Link)</li><li>Verify true/false toggle, defaulted to true and help text</li><li>Set to true and Save</li><li>ALSO: Observe the URL in Help Link field</li><li>AND: Observe the URL in Report a Problem Link field</li></ol></article>

**Test Data**

> <article><img src="https://smartbear-tm4j-prod-us-west-2-attachment-rich-text.s3.us-west-2.amazonaws.com/embedded-f3277290f945470c4add5d21ef3dc7ca7b74388fc7152bfb6b99ae58c66a95a8-1602046661070-Screen+Shot+2020-10-06+at+9.54.38+PM.png" style="width: 340px;" class="fr-fil fr-dib"></article>

**Expected**

> <article><ul><li>Help Text:<br>When true, 'Ask the community' link appears on the Mattermost user interface and Main Menu, which allows users to join the Mattermost Community to ask questions and help others troubleshoot issues. When false, the link is hidden from users.</li><li>Setting should be enabled</li></ul></article>

<hr/>

**Step 2**

> <article><ol><li>Hover over Help icon</li><li>Click on the Help icon</li><li>After enabling the setting, verify Ask community link is available under Help icon menu</li></ol><br></article>

**Test Data**

> <article><img src="https://smartbear-tm4j-prod-us-west-2-attachment-rich-text.s3.us-west-2.amazonaws.com/embedded-f3277290f945470c4add5d21ef3dc7ca7b74388fc7152bfb6b99ae58c66a95a8-1596829735594-Screen+Shot+2020-08-07+at+3.47.43+PM.png" style="width: 300px;" class="fr-fil fr-dib"></article>

**Expected**

> <article><ul><li>Help icon (question mark within circle) shows "Help" text on mouse hover</li><li>Help menu should show 4 options<ul><li>Ask the community</li><li>Help resources</li><li>Report a problem</li><li>Keyboard shortcuts</li></ul></li></ul></article>

<hr/>

**Step 3**

> <article><ol><li>Click on Ask the community link</li></ol></article>

**Expected**

> <article>Link opens https://mattermost.com/ask-mattermost-community/ (if you're not logged in or redirects directly to https://community.mattermost.com/core/channels/peer-to-peer-help if you are logged in)<img src="https://smartbear-tm4j-prod-us-west-2-attachment-rich-text.s3.us-west-2.amazonaws.com/embedded-f3277290f945470c4add5d21ef3dc7ca7b74388fc7152bfb6b99ae58c66a95a8-1596830099053-Screen+Shot+2020-08-07+at+3.54.42+PM.png" style="width: 300px;" class="fr-fil fr-dib"></article>

<hr/>

**Step 4**

> <article><ol><li>Disable the setting for Ask community<br>(Mark false in sys console and Save)</li></ol></article>

**Test Data**

**Expected**

> <article><ul><li>Help menu should show 3 options<ul><li>Help resources</li><li>Report a problem</li><li>Keyboard shortcuts</li></ul></li></ul></article>
