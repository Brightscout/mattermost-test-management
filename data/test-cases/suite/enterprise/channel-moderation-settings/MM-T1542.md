### Steps

<hr/>

**Step 1**

> <article><ol><li>Login as sysadmin and visit the Channels page and click on a channel.</li><li>Uncheck the Create Posts option for Members and save.</li><li>Note: The option to create posts for Guests should still be allowed.</li><li>Login as a Regular Member and visit the same channel.</li><li>Check if the user has the permission to create a post on that channel.</li><li>As a sysadmin, check the option to allow Create Posts for Members and save.</li><li>Check if the user has the permission to create a post on that channel.</li></ol></article>

**Expected**

> <article>After #5,<br><ul><li>Member should not have the permission to create a post on a channel when the option is removed.</li></ul>After #7,<br><ul><li>Member should have the permission to create a post on a channel when the option is allowed.</li></ul><br></article>
