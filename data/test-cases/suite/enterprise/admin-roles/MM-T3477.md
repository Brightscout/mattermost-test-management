### Steps

<hr/>

**Step 1**

> <article>1. Login as Sysadmin and visit System Console &gt; System Roles page.<br>2. Click on each of the following roles and check if the User Management &gt; System Roles permission is always set to "No Access" and sysadmin should not be allowed to change this dropdown and it should not be editable.<ul><li>System Manager</li><li>System User Manager</li><li>Read Only Admin</li></ul>3. Click on the permissions dropdown for User Management and select "Can Edit" and notice that the System Roles permission should remain as "No Access"</article>

**Expected**

> <article>System Roles permission should not be assignable to any of the System Roles and should always be "No Access". Only sysadmin should have this permission enabled&nbsp;</article>
