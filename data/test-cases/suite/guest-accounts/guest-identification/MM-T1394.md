### Steps

<hr/>

**Step 1**

> <article>Manage Users - Change Email not whitelisted for Guest user<br>–––––––––––––––––––––––––<ol><li>Login to the application and click on Invite People and invite few guests. Accept all the invitations and ensure there are different guest users in the system.</li><li>Now Log in as a team admin and navigate to System Console ➜ Guest Access Page.</li><li>Add a domain in whitelisted domain section.</li><li>Now navigate to users page and select a Guest User and click on Change Email.</li><li>Try to enter an email which does not belongs to the whitelisted domain and check if Admin is allowed to enter an email which is not whitelisted.</li></ol></article>

**Expected**

> <article><ul><li>Admin should not be allowed to enter an email for a Guest User which does not matches the whitelisted domain. An error should be shown to the Admin and this option should be restricted from the System console.</li></ul></article>
