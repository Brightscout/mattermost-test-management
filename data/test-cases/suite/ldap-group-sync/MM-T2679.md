### Steps

<hr/>

**Step 1**

> <article>Channel Role ignores Guest Users<br>————————————————————————————<ol><li>Set Guest Filter in LDAP Configuration page such that some users are marked as Guest.</li><li>Login as LDAP users matching the Guest Filter and ensure that their account is created as a Guest.</li><li>For a new Channel, select a Group and set the Role to Channel Admin in System Console ➜ Channels page.</li><li>Check if all users who are not Guest Users have Channel Admin permissions.</li><li>Guest Users should have only Channel Guest permissions and should not be promoted to Channel Admin.</li></ol></article>

**Expected**

> <article><ul><li>Guest Users should have only Channel Guest permissions and should not be promoted to Channel Admin.</li></ul></article>
