### Steps

<hr/>

**Step 1**

> <article>Team Role ignores Guest Users<br>————————————————————————————<ol><li>Set Guest Filter in LDAP Configuration page such that some users are marked as Guest.</li><li>Login as LDAP users matching the Guest Filter and ensure that their account is created as a Guest.</li><li>For a new Group Synced Team, select a Group and set the Role to Team Admin in System Console ➜ Teams page.</li><li>Check if all users who are not Guest Users have Team Admin permissions for that team.</li><li>Guest Users should not be added to the team.</li></ol></article>

**Expected**

> <article><ul><li>Guest Users should not be added to the LDAP Group Synced Team unless they are added to a specific channel in the team</li></ul></article>
