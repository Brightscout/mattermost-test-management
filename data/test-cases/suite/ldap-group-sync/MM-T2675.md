## MM-T2675: Demote Channel Role Permission

### Details

### Steps

<hr/>

**Step 1**

> <article>Demote Channel Role Permission<br>————————————————————————————<ol><li>For a Group Synced Channel, select a Group and set the Role to Channel Admin in System Console ➜ Channels page.</li><li>Run LDAP Group Sync job and check if user has access to the Channel with Channel Admin permission.</li><li>Change the role from Channel Admin to Members for the same channel in System Console ➜ Channels page.</li><li>User should immediately be updated to Channel Member role and should not be able to perform Channel Admin tasks like view and modify Channel Members, Manage Groups etc.</li><li>Run LDAP Group Sync job again and check if user User still has Channel Member permission for that Channel.</li></ol></article>

**Expected**

> <article><ul><li>User should not be able to perform Channel Admin tasks like downgrade other user's permissions</li></ul></article>
