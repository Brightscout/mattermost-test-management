### Steps

<hr/>

**Step 1**

> <article>Get ID values from web<br>------------------------<br>1. Chrome: More Tools &gt; Developer Tools &gt; Network<br>2. If logged in MM webapp, refresh page<br>3. Tap on Name column to sort and look for <code><strong>me</strong></code> entry or any entry that would contain cookies<br>4. Cookie values,<ul><li><strong>MMUSERID</strong> = User Actual ID</li><li><strong>rl_user_id</strong> or <strong>ajs_user_id</strong> (without the <code>%22</code>) = Server ID</li></ul>5. Take note of the values<br><br><br></article>
