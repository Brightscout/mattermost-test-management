> <article>On 2. Both options to execute are shown: executing command or input options as part of commands<br />On 5. The data is validated. OpsGenie App bot returns an ephemeral post as a response:  <br />- If the alert ID is valid, the alert's note will be added <br />- If not, the post shows the error</article>
