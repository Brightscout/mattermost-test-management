### Steps

<hr/>

**Step 1**

> <article><u>Open an unfollowed thread</u><ol><li>As User1, create new root post in Off topic <ol><li>Verify for User1 this channel is <strong>not</strong> bolded</li><li>Verify channel item <strong>doesn't</strong> have mention badge</li><li>Verify global threads item is <strong>not</strong> bolded</li><li>Verify global threads item <strong>doesn't</strong> have mention badge</li><li>Verify main channel root post <strong>doesn't</strong> have a thread footer and <strong>no</strong> unread dot</li><li>Verify <strong>no</strong> new messages line on the channel</li></ol></li><li> Login as User2 and mark Off topic as read</li><li>As User2 switch to different channel<ol><li>Verify Off topic is <strong>not</strong> bolded</li></ol></li><li>As User1 add replies to post from step 1.<ol><li>Verify, for user1, root post has a thread footer, Following and <strong>no</strong> unread dot</li></ol></li><li>As User2 click on Off Topic to open <ol><li>Verify Off topic is <strong>not</strong> bolded</li><li>Verify global threads item is <strong>not</strong> bolded</li><li>Verify main channel post from step one has a thread footer, Follow and <strong>no</strong> unread dot</li><li>Verify <strong>no</strong> new messages line above the root post</li></ol></li><li>Click on the root post to open on RHS <ol><li>Verify Follow button</li><li>Verify <strong>no</strong> new message line</li></ol></li></ol></article>

**Expected**

> <article>Reading unfollowed threads should have no effect on badge/bolding behavior for the channel sidebar or thread sidebar items</article>
