### Steps

<hr/>

**Step 1**

> <article>Profile picture should read labels<br />–––––––––––––––––––––––––<ol><li>Open Profile &gt; Profile Settings and tab down to the Edit button from Profile picture and enter</li><li>Tab through the buttons and hit enter when Select button has the focus.</li><li>Upload a picture and tab through the buttons and hit enter when Save button has the focus.</li><li>Close the dialog and open Profile Settings again and tab down to edit button from profile picture and enter.</li><li>The focus should be on the Remove profile picture and screen reader should read the button 'Remove profile picture'</li><li>Hit Enter and the profile picture should be removed</li><li>Tab through the buttons and the screen reader should read: 'Select button', 'Save button' and 'Cancel button'</li></ol></article>

**Expected**

> <article><ul><li>Upload dialog should be shown when enter key is pressed when select button has the focus</li><li>When a profile picture already exists and edit button from profile picture section is entered, then the focus should be on the Remove profile picture and the screen reader should read: 'Remove profile picture'</li><li>When TAB is pressed, the screen reader should read the buttons: 'Select button', 'Save button' and 'Cancel button'</li></ul></article>
