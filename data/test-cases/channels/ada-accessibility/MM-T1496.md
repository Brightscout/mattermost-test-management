### Steps

<hr/>

**Step 1**

> <article>Security Settings screen should read labels<br />–––––––––––––––––––––––––<ol><li>Open User Profile  &gt; Profile modal and select Security.</li><li>Press TAB key and focus on Edit button of Multi factor authentication.</li><li>Focus should be on "Add MFA to your account" button.</li><li>Hit Enter and it should open the step to configure MFA.</li><li>Go back and focus on Edit button of OAuth 2.0 Applications.</li><li>If OAuth application is configured, focus should be on first application.</li><li>Focus on Edit button of Sign-in Method and hit enter.</li><li>Focus should be on the first button 'Switch to using Google SSO"</li></ol></article>

**Expected**

> <article><ul><li>The screen reader should highlight the first option under the security setting once it's expanded with the "edit button". ie Multi-factor auth, OAuth 2.0 apps, sign-in method</li></ul></article>
