### Steps

<hr/>

**Step 1**

> <article>Manage Channel Members<br>–––––––––––––––––––––––––<ol><li>Click on Manage Members from a Channel Menu</li><li> Press TAB to navigate between different Remove Member buttons</li><li> If the Remove Member button is not present, it should have Channel Member buttons instead</li><li> Click on search box and type a text such that it yields no results</li><li>&nbsp;Click on search box and type a letter such that it yields results and then press TAB</li></ol></article>

**Expected**

> <article><ul><li>When Manage Members dialog is opened, ensure the reader reads the title as "Dialong Manage Members" and the name of the channel</li><li>Search box should have focus and should be read after the dialog name</li><li>Tab through to different Remove Channel Member buttons should read the "[Username] remove member button"</li><li>If Remove Channel button is not present, it should have Channel Member buttons instead and should read "[Username] channel member button"</li><li>When searched for users without any results, it should state "0 of X members total"</li><li>When searched for users with results, it should state "N of X members total"</li><li>When searched for users with results and TAB is pressed, it should read the username of the first user in the list and the name of their current role.</li><li>Pressing TAB keys should focus between the 'Remove Member' buttons of different users. The focus should not go automatically back to the search input</li></ul></article>
