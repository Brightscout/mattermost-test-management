## MM-T1538: Download a video

---

**Step 1**

1. Post a video
2. Open the desktop app where the video file was shared/posted on
3. Click on the Download button
4. Select a file download location
5. Click Save 

**Test Data**

![](https://smartbear-tm4j-prod-us-west-2-attachment-rich-text.s3.us-west-2.amazonaws.com/embedded-f3277290f945470c4add5d21ef3dc7ca7b74388fc7152bfb6b99ae58c66a95a8-1590773241431-Screen+Shot+2020-05-29+at+1.27.05+PM.png)

**Expected**

- File downloads successfully
- Verify Download Complete popup notification is present 
- The Desktop app doesn't crash

---

**Step 2**

1. Click on the Download Completed popup

**Expected**

- Saved file location selected in 2nd step opens in the 
- File Explorer (or Finder in MacOS) with the file selected

---

**Step 3**

1. Create a folder on your computer and set permission to **Read-Only** for all users
2. Click on a file to download again and select this read-only folder as a destination folder
3. Click Save

**Expected**

- App should not crash when download fails.\
  \
  **Note**: Currently on macOS the only feedback user receives for failed download is the lack of Download Completed notification popup

---

**Step 4**

1. Right click on an image shown in Mattermost (such as a profile image or the thumbnail for a file attachment)
2. Select "Save Image"
